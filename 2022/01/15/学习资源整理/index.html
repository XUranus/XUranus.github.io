学习资源整理

发表于 2022-01-15 更新于 2022-08-14 Disqus： 598 1 分钟

这个时代要学个东西网上资源丰富而又良莠不齐，映像里大多数培训班直播喜欢东拉西扯视频质量贼差，而有些用爱发电的UP做的教程往往PPT做的特别精致。

这里搜集一些本人学过的精品资源，每一个都是宝藏。

408基础

数据结构

清华大学邓俊辉数据结构与算法 34h，C++模板编程描述数据结构，内容相当丰富。

语言类

C++

  • B站LH_mouse，STL源码解析，手撮轮子看他
  • C++中文网，快速入门教程

Java

Rust

B站UP主软件工艺师，微软MVP，一股带点东北口语的标准普通话，咬字清楚，认真做PPT，全程无废话，听他讲课真的舒服!

  • Rust编程语言入门教程（Rust语言/Rustlang）。 完全是按照Rust官方的《The Book》来讲课的，看《The Book》困难的可以看他的视频。

工具类

Git

  • 廖雪峰Git教程，我按照他的教程过完一遍Git基本操作流程，还写过一篇博客。完全基于实战去解释Git的操作，感觉很适合没接触过Git的人看，他的教程算是我认为新手入门Git的最佳捷径。

存储

张书宁，前华为存储的大佬。B站SunnyZhang的IT世界，投稿大多涉及文件系统底层相关知识

  • 《文件系统技术内幕》系列，介绍了Linux文件系统相关原理

知乎itworld123，《文件系统技术内幕》的作者在知乎的帐号，关注其专栏和回答

  • Linux文件系统彻底解析
  • 数据存储技术
  • Linux内核开发从入门到精通

资源网

  • 52doc
  • zlib.cc
