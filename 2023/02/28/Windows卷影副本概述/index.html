Windows卷影副本概述

发表于 2023-02-28 更新于 2023-03-11 Disqus： 2.2k 2 分钟

Windows Server 2008以上的版本，管理员可以通过卷影副本服务（VSS）在指定的卷上创建备份的副本。副本类似于快照(Snapshot)，可以基于快照将卷恢复到之前某个版本，也可基于快照做一致性备份。因此，它可以极大保护服务器的数据安全。

项目完整地址见https://github.com/XUranus/Win32VSSWrapper

卷操作

使用cmd管理卷影副本

使用管理员权限进入cmd
卷影副本命令vssadmin用于在命令行管理卷影副本

列出所有卷影副本

默认情况下，Windows Server 2008的各卷是启动了卷影副本的。
使用vssadmin add shadowstorage /for=<Volume1> /on=<Volume2>卷影副本。<Volume1>指定了要保护的卷，<Volume2>指定了副本存放的卷。副本存放的卷可以是要保护的本地卷，也可以是其他卷。
